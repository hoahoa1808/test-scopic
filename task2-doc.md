## Instruction 

* **Backend Side**

  - create environment
    ```
    cd webUI/backend
    conda create --name backend
    conda activate backend
    pip install -r requirements.txt
    ```
  - then, run server by:
    ```bash
    cd webUI/backend
    conda activate backend
    python app.py
    ```
